## 게임 소개

AI NPC들 사이에 숨어있는 플레이어들을 찾아내는 숨바꼭질과 조준 사격이 결합된 멀티플레이 게임입니다.

## 게임 방법

1. **숨바꼭질 페이즈** — AI NPC들 사이에 섞여 다른 플레이어를 찾아 조준합니다.
2. **탑뷰 전환** — 타이머 종료 시 카메라가 탑뷰로 전환되고 AI NPC가 사라집니다.
3. **사격** — 모든 생존자의 조준선이 동시에 공개되고 각자 조준한 방향으로 발사합니다.
4. 한 명만 남을 때까지 반복합니다.

## 조작법

| 키 | 동작 |
|---|---|
| WASD | 이동 |
| Shift | 달리기 |
| 마우스 | 카메라 |
| ESC | 메뉴 |
